¿Qué es Frontend?

El término frontend se refiere a la parte de una página web o aplicación que los usuarios pueden ver y con la que interactúan directamente. Es la interfaz gráfica de usuario (GUI), donde se muestran los contenidos y se gestionan las interacciones.

Fundamentos del Frontend

El desarrollo frontend se basa principalmente en tres tecnologías fundamentales: HTML, que define la estructura del contenido; CSS, que controla la presentación visual y el diseño; y JavaScript, que permite agregar interactividad y dinamismo a la página.
