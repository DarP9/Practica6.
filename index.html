¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar que se utiliza para estructurar el contenido de las páginas web.

¿Qué es CSS?

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje que se utiliza para dar formato y estilo a los documentos HTML.

¿Qué es JAVASCRIPT?

JavaScript es un lenguaje de programación que se utiliza principalmente en el desarrollo web para hacer que las páginas sean interactivas y dinámicas.
